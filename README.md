1.What is the difference between var, let, and const?
Ans:Var is hoisted and initialized as undefined.It is function-scoped.let and const are hoisted but not initialized.They are block-scoped.

2. What is the spread operator (...)?
Ans:The spread operator (...) in JavaScript is used to expand or spread elements of an array, object, or iterable into individual elements
